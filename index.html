Summary: Latitude vs. X

The purpose of this project was to determine how proximity to the equator (latitude) affects weather changes. To accomplish this analysis, I first created a data set by pulling information from OpenWeatherMap API, then used Pandas to assemble a Data Frame of the 500+ cities world wide.

I was then able to use Matplotlib to plot and compare latitude with different weather variables. Factors I looked at included: maximum temperature, humidity, cloudiness, and wind speed. This site provides the source data and vizualizations created for the analysis, as well as explainations and descriptions of any trends I noticed.
